Cron-Insta

La herramienta que te deja escribir

Un editor de escritorio para novelistas. Sin menús, sin distracciones, sin nube. Todo en tu disco: capítulos, personajes, notas, línea de tiempo y lugares. Git invisible para que nunca pierdas una palabra.

Tres principios, una sola obsesión

Cron-Insta está diseñado para que te olvides de la herramienta. No es un procesador de textos: es el espacio donde tu historia cobra vida.

01

Zona de escritura limpia

Sin barras de herramientas, sin menús flotantes, sin formatos que interrumpan. La pantalla es para tus palabras.

02

Todo en tu disco

Archivos Markdown y JSON en una carpeta que tú controlas. Sin nube, sin registro, sin dependencia externa. Tus palabras son tuyas.

03

Git invisible

Cada vez que cierras la aplicación se crea un punto de control. Historial completo con conteo de palabras sin que tengas que pensar en ello.

Todo lo que tu novela necesita

El panel lateral concentra las herramientas de organización. Aparecen cuando las necesitas y desaparecen cuando escribes.

Capítulos

Crea, ordena y navega entre capítulos con el teclado. Cada capítulo es un archivo independiente en tu proyecto.

Fichas de personajes

Descripción física, personalidad, historia, relaciones entre personajes. Fichas completas que crecen con tu historia.

Notas

Ideas sueltas, investigación, esbozos. Notas enriquecidas con el mismo editor que usas para tus capítulos.

Línea de tiempo

Eventos del presente y del pasado, vinculados a personajes, capítulos y lugares. Arrastra y suelta para reordenar.

Lugares

Cada escenario de tu novela tiene su ficha: nombre, descripción, y vinculación automática con los eventos de la línea de tiempo.

Exportación y respaldo

Proyecto completo en .zip para respaldo, o todos los capítulos en un solo .md para compartir con lectores beta.

🧠 Preparado para IA

Cada proyecto incluye un archivo SCHEMA.md que describe el modelo de datos completo: entidades, relaciones, rutas de archivos. Diseñado para que herramientas de IA puedan entender y navegar tu proyecto sin esfuerzo.

Empieza en segundos

Sin registro, sin configuraciones complicadas, sin cuentas. Abres la aplicación y escribes.

1

Crea tu proyecto

Elige carpeta, nombre y tipografía. Cron-Insta crea la estructura de directorios y archivos. Si tienes Git instalado, también inicializa el control de versiones.

2

Escribe sin pensar en la herramienta

El editor ocupa casi toda la pantalla. Guardado automático cada 20 segundos. El panel lateral aparece solo cuando lo necesitas.

3

Organiza tu mundo

Fichas de personajes con relaciones, notas de investigación, línea de tiempo con eventos vinculados a lugares y personajes. Todo conectado.

Nunca pierdas una palabra

Git integrado de forma invisible. Cada sesión queda registrada con un punto de control automático y un conteo de palabras.

  • Punto de control automático al cerrar la aplicación
  • Conteo de palabras en cada punto de control
  • Historial completo de sesiones desde el panel de herramientas
  • Funciona igual si Git no está instalado — sin errores ni avisos
  • Sincronización remota opcional por SSH (GitHub, GitLab)
  • Detección automática de repositorios clonados: reconoce tu identidad y remoto sin configurar nada
  • Protección de 3 intentos: si la sincronización falla, se desactiva sola para no interrumpir tu escritura
